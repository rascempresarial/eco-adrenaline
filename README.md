# Eco Adrenaline, Landing Page

Landing page da Eco Adrenaline, agência de turismo de aventura em Maringá, PR.
Site estático de arquivo único, pronto para GitHub Pages.

## Estrutura
- `index.html`, página completa (HTML, CSS e JS embutidos)
- `assets/logo-eco-adrenaline.svg`, logo vetorial
- `robots.txt` e `sitemap.xml`, suporte a SEO

## Seções
Topo com logo centralizada em barra branca, Agenda, Sobre Nós, Passeios, Últimos Posts (carrossel) e Fale Conosco (WhatsApp).

## Publicar no GitHub Pages
1. Crie um repositório no GitHub e suba estes arquivos na raiz.
2. Em Settings, Pages, selecione a branch `main` e a pasta `/root`.
3. O site fica disponível em `https://SEU-USUARIO.github.io/NOME-DO-REPO/`.

### Domínio próprio (opcional)
Para usar `ecoadrenaline.com.br`, crie um arquivo `CNAME` com o domínio dentro e configure o DNS apontando para o GitHub Pages. Caso o domínio mude, atualize as URLs em `index.html` (canonical e dados estruturados), `robots.txt` e `sitemap.xml`.
